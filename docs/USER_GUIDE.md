# Dream Flow Creator Studio - User Guide

Welcome to Dream Flow Creator Studio! This guide will help you create, edit, and manage your AI-generated sleep stories.

## Getting Started

### Creating an Account

1. Open the app and tap "Sign Up"
2. Enter your email address and create a password
3. Verify your email (check your inbox)
4. Log in with your credentials

### First Story

1. Navigate to the **Generator** tab
2. Enter a story prompt (e.g., "A calming story about a peaceful forest")
3. Select a theme (optional)
4. Adjust parameters:
   - **Tone**: peaceful, adventurous, mysterious, inspiring, dreamy
   - **Length**: short, medium, long
   - **Style**: narrative, descriptive, poetic
5. Choose media options:
   - Enable "Generate Video" for video version
   - Enable "Generate Audio" for audio narration
6. Tap "Generate Story"

Your story will be created and you'll be taken to the editor.

## Features

### Story Generator

The generator uses AI to create custom sleep stories based on your prompts.

**Tips for Better Stories:**
- Be descriptive in your prompts
- Include mood and setting details
- Specify the type of story you want (adventure, meditation, etc.)

**Themes Available:**
- Nature
- Fantasy
- Calming
- Adventure
- Meditation
- Space
- Ocean
- Forest
- Mountain
- Desert

### Story Editor

The editor lets you refine and customize your generated stories.

**Features:**
- **Auto-save**: Changes are automatically saved every 2 seconds
- **Undo/Redo**: Use the toolbar buttons to undo or redo changes
- **Formatting**: 
  - Bold text: Select text and tap Bold button
  - Italic text: Select text and tap Italic button
  - Headings: Select text and tap Heading button
  - Lists: Tap List button to create bullet lists
- **View Modes**:
  - **Edit**: Focus on writing
  - **Preview**: See how your story looks
  - **Split**: Edit and preview side-by-side
- **Word Count**: See word and character count at the bottom

**Export Options:**
- Tap the "Export" button in the header
- Choose format: PDF, Markdown, or JSON
- File will download to your device

### Library

Your library contains all your stories in one place.

**View Modes:**
- **Grid View**: See stories as cards (tap grid icon)
- **List View**: See stories in a list (tap list icon)

**Search:**
- Use the search bar to find stories by title or content
- Search is case-insensitive and searches both title and content

**Filters:**
- Tap the filter icon to open filters sidebar
- Filter by:
  - **Theme**: Select specific themes
  - **Media**: Show only stories with video/audio
  - **Sort By**: Date created, last updated, or title
  - **Sort Order**: Ascending or descending

**Bulk Operations:**
1. Long-press a story to select it
2. Select multiple stories
3. Use bulk actions:
   - **Export**: Export multiple stories (coming soon)
   - **Delete**: Delete selected stories
   - **Select All**: Select all visible stories

**Story Actions:**
- **Edit**: Tap story card to open in editor
- **Export**: Tap export icon on story card
- **Delete**: Tap delete icon on story card

### Preview Screen

The preview screen shows your story with media playback.

**Media Status:**
- **Pending**: Media generation queued
- **Processing**: Media is being generated
- **Ready**: Media is available for playback
- **Failed**: Generation failed (tap Retry to regenerate)

**Playback:**
- **Video**: Use native video controls
- **Audio**: Tap play button to start audio narration

**Regeneration:**
- If media generation fails, tap "Retry" on the status indicator
- Media will be regenerated with the latest story content

### Analytics Dashboard

Track your creative progress and story statistics.

**Metrics Shown:**
- **Total Stories**: Number of stories you've created
- **With Media**: Stories that have video or audio
- **Average Length**: Average word count of your stories
- **Last 7 Days**: Stories created in the past week

**Media Generation Stats:**
- Video success rate
- Audio success rate
- Total attempts and successful generations

**Popular Themes:**
- See which themes you use most often
- Ranked by frequency

**Usage Over Time:**
- Chart showing stories created over time
- Adjust date range to see different periods

**Export Statistics:**
- Track how many times you've exported stories
- Breakdown by format (PDF, Markdown, JSON)

**Date Range Selection:**
- 7 Days: Last week
- 30 Days: Last month
- 90 Days: Last quarter
- All Time: Complete history

## Tips and Best Practices

### Creating Great Stories

1. **Be Specific**: Detailed prompts create better stories
2. **Use Themes**: Themes help guide the AI's style
3. **Experiment**: Try different tones and styles
4. **Edit After Generation**: Always review and refine AI-generated content

### Managing Your Library

1. **Use Search**: Quickly find stories by keywords
2. **Organize by Theme**: Use theme filters to group similar stories
3. **Regular Cleanup**: Delete stories you no longer need
4. **Export Regularly**: Export important stories for backup

### Media Generation

1. **Be Patient**: Media generation can take time
2. **Check Status**: Monitor media status in preview screen
3. **Retry on Failure**: If generation fails, try regenerating
4. **Consider Costs**: Video generation may have usage limits

### Exporting Stories

1. **Choose Format**: 
   - PDF: Best for sharing and printing
   - Markdown: Best for editing in other tools
   - JSON: Best for data backup and integration
2. **Include Metadata**: Adds creation date, theme, and other info
3. **Include Media Links**: Adds video/audio URLs to export

## Troubleshooting

### Story Generation Fails

- Check your internet connection
- Verify you have sufficient credits/quota
- Try a different prompt
- Check error message for details

### Media Generation Stuck

- Check media status in preview screen
- Wait a few minutes (generation can take time)
- Try regenerating if stuck on "processing"
- Check if Redis/queue service is running (technical)

### Export Fails

- Check file permissions on your device
- Ensure you have storage space
- Try a different format
- Check error message for details

### Can't Find Stories

- Use search to find by keywords
- Check filters aren't hiding stories
- Verify you're logged into the correct account
- Try refreshing the library (pull down)
